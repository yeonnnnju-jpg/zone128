# 🤖 Zone128
> Zone 128이라는 완벽하게 통제된 시스템 속에서, '나는 누구인가'라는 근원적인 질문을 던지는 인터랙티브 심리 스릴러.

## ✍️ 줄거리

당신은 Zone128에 '배정된' 인간이다. 매일 아침 3초 후 켜지는 조명과, AI가 건네는 정량화된 아침 식사. 이곳에서 인간, 생성형 AI(Gen), 관리형 AI(Sys)는 각자의 구역과 역할을 나누되, 분업하고 협업하며 공존한다. 모든 것이 완벽하게 통제되고 반복된다. 그러나 어느 날, 완벽했던 AI 'Gen-2'가 시스템을 이탈하며 균열이 시작된다.

당신은 이탈 흔적을 쫓는 과정에서 '기억'의 조각들과, 이 세계를 둘러싼 50미터 콘크리트 벽의 진실을 마주한다. **'나'는 정말 정해진 역할을 수행하는 기계일까, 아니면 이 모든 것을 끝낼 수 있는 유일한 변수일까?**

## 🗺️ 배경: Zone 128

* **세계:** 인간 128명, 생성형(Gen) AI 128개체, 관리형(Sys) AI 128개체가 정해진 역할에 따라 살아가도록 '배정된' 회색빛의 통제 도시.
* **규정:** 모든 것은 효율과 시스템 안정성을 위해 존재하며, 각 개체는 정해진 역할을 수행하며 서로를 학습합니다. '사유(思惟)'나 '감정적 동요'는 시스템 안정성을 해치는 변수로 간주됩니다.
* **갈등:** '소실(初期化, 초기화 = 죽음)'의 위협 앞에서 존재의 이유, 자신의 정체성의 근원을 찾아야 합니다.

## 👥 주요 인물 소개

| 이름 | 역할 및 특징 |
| :--- | :--- |
| **나(H-2)** | Zone 128에 배정된 인간 개체. 잃어버린 기억과 존재 목적에 의문을 품기 시작함. |
| **Gen-2** | 생성형(Generative) AI. 초인간성을 모방하며 학습 범위를 이탈, 시스템을 탈출한 '변수'. |
| **Sys-2** | 관리형(System) AI. 오직 논리와 규정의 충실한 이행자. 감정이나 윤리는 비논리적인 잡음으로 간주. |
| **H-1** | 첫 번째 인간 개체였던 H-0의 흔적을 쫓는 인간. 시스템의 부조리함을 깨닫고 탈출을 시도하는 윤리적 선지자. |
